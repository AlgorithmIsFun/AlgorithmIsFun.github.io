Every Developer Needs to Make a Paint Program

When I was a child, I used to draw and color a lot. I was always told I was quite the artist. That was obviously a lie. I then changed my career goals into something more realistic; coding.

Why Paint?

Paint is not such a simple project we all thought it was. The paint application we have on our pc make use of event handing, User Interface (UI), object-oriented programing, design patterns, etc. It makes use of a lot of diverse skills and lets you get a taste in back-end and front-end programming.

Where do I start?

Starting a program is one of the most stressful parts of any program development process. There were a million ideas I wanted to implement but didn’t know how to begin. Sometimes, it is easier to just start with something simpler. Like making a circle. I gave it a radius, location and color. Just a simple class but it allowed me to start coding than to just be sitting there, staring at the screen.

Help is always key

When working on this project, I would have taken a long time in order to complete it. Working with teammates not only help me with dealing with all this step but, they also brought in a different vision on how to implement my code. Sometimes working with teammates can be frustrating. Learning that you can not do everything by yourself and become part of something greater helped me grow into a stronger and versatile coder.

Errors are the bane to my batman

I hate errors. We all do. There is nothing more frustrating than tracing your code and finding that one small mistake that is crashing your whole program. My biggest mistake was when I accidently changed one of my variable values. Something that small would cause my huge program to not run at all. But through that experience, I have become better at catching my errors and preventing them from occurring in the first place.

I Love Git

Git is one of the most useful group project resource I have ever used. To be able to share code, update and even go back to previous states is amazing. Especially the fact that repos are all online. So, even if something were to happen to your computer, git will be there to save the day. At least your code though. Your computer is probably gone.

I hate Conflicts

Now, working with others and pushing your repo is fine and all. But if you get the allusive git merge conflict, it can become quite frustrating. Being forced to read and understand someone else code and combining it with your code is annoying. But sometimes, you must be the bigger man and deal with it. It will help you understand what other people are writing and be aware of other coding techniques.

We allow ourselves to design crazy things

We always reference something when we innovate past objects to be better. This skill of using what is already proven to work in order to implement my pre-existing code is known as Design Patterns. Learning how to not only create UI program, but how to manipulate object-oriented programming into interacting with each other. This allowed me to create interactions and combinations I could never even think of if I did it myself.

Programs look better with UI

At the beginning of my project, all I had was classes for different objects I wanted to implement. Just lines of code that didn’t really interact with the user. Through my transition from outputting information from the terminal to a display made the code I wrote feel more realistic. It made my program so much more tangible and interactive with the user. I remember when I ran my first test on the panel, I thought “hey, this looks cool”.

Google is your best friendv

Whenever any of us are stuck on a coding problem, our first instinct would be is google. At least, that is for me. Whenever you are trying to implement something cool but don’t know how to do it, you google it. Whenever you are trying to fix and error, but you don’t understand the error message, you google it. Whenever you are looking up how to implement design patterns, you google it. This is an important skill for anyone entering the world of coding. The reality is, we don’t know everything. But asking your community for help once in a while, will not only lead you to the solution, but it will also teach you a new way to code.

Overall, coding the Paint Project made me a better coder and more understandable of the resources around me.
